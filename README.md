# PharmacyApp

This is MERN stack web project

# Run Locally

1. Clone repo
2. Setup MongoDB
   Install it from here [Click here](https://downloads.mongodb.com/windows/mongodb-windows-x86_64-enterprise-5.0.2-signed.msi) 
3. Run Backend
  * $ cd backend
  * $ npm install
  * $ npm start
4. Run Frontend
  open new terminal
  * $ cd frontend
  * $ npm install
  * $ npm start
